PMA packagesmanagerAD-coding

Это официальное хранилище пакетов и плагинов для **AsmDone**.

### Как это работает:
IDE автоматически подключается к этому репозиторию через встроенный менеджер пакетов (PMA). 

### Для разработчиков:
Если вы хотите добавить свой пакет:
1. Загрузите `.ad` файл в папку `packages/`.
2. Обновите `packages.json`, добавив данные о вашем пакете.
3. Отправьте Pull Request.
